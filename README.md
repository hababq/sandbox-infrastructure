# sandbox-infrastructure

## Overview
This project implements a fast sandbox infrastructure for testing applications in isolated environments.

## Features
- Lightweight sandbox setup
- Easy to use and deploy
- Supports multiple environments

## Tech Stack
- Docker
- Kubernetes

## Usage
1. Clone the repository.
2. Install dependencies: `npm install`
3. Deploy the sandbox: `docker-compose up`
